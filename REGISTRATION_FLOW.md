## 🤖 AI가 자동으로 하는 일 (사장님은 보기만)

### Step 1: 원문 이해 (30초)
- Gemini AI가 원문을 읽어 구조화
- 내용: 상품명, 목적지, 일정, 가격, 호텔, 관광지, 선택관광, 유의사항 등 **40여 개 필드 추출**
- 만약 "메르데카 광장 같은 원문에 없는 내용"이 파싱되면 → **자동 차단**

### Step 2: 검증 (5초)
- **19가지 체크리스트** 자동 실행 (W1~W19)
- 실패 시: `draft` 상태로 격리 (고객에게 안 보임)
- 통과 시: 바로 다음 단계

**주요 검증 항목 (W16~W19 - 신규)**:
- ✅ 출발요일이 `["금"]` 같은 JSON 누출 아닌가?
- ✅ 선택관광에 "2층버스 (싱가포르)" 같이 지역 라벨 있는가?
- ✅ 원문에 없는 관광지(메르데카 광장 등)를 임의로 삽입했는가?
- ✅ 일차 수 맞는가? (4박6일 → 6일)

### Step 3: DB 저장 (5초)
- Supabase에 INSERT
- 중복 상품 자동 감지 → 겹치면 이전 버전 아카이브

### Step 4: 화면 즉시 갱신 (1초)
- ISR 캐시 자동 무효화
- **모바일 랜딩이 1시간 대기 없이 바로 반영** (이전 개선)

### Step 5: 🆕 원문 ↔ 렌더 자동 대조 감사 (20초)
등록 직후 자동 실행:

체크 항목 (자동):
- 원문의 **모든 가격** (1,249,000 등)이 화면에 있는가?
- 원문의 **모든 호텔명**이 화면에 있는가?
- 원문의 **모든 관광지 (▶표시 항목)**가 화면에 있는가?
- 원문의 **모든 항공편**이 화면에 있는가?

4. 어드민에서 **[승인] 버튼** 클릭 → 자동으로:
   - 고객 노출 시작
   - `baseline_requested_at` DB 설정 → GitHub Action이 2시간 내 Visual baseline 자동 생성
   - 이후 모든 코드/데이터 변경이 회귀 탐지됨

### 가끔 해야 하는 일 (월 1~2회)
5. **이슈 리포트 나오면** 어드민 링크 클릭해서 수정 (1~2분)
6. **GitHub Issue 가 올라오면** (회귀 탐지됨) 확인 후 수정 or 의도적 변경이면 "accept changes" 액션 실행

### 절대 안 해도 되는 일 (자동화됨)
- ~~`npm run test:visual:update`~~ → GitHub Action `baseline-refresh.yml` 이 2시간마다 자동 실행
- ~~`npm run test:visual`~~ → GitHub Action `visual-regression.yml` 이 매일 새벽 3시 자동 실행
- ~~`curl /api/revalidate`~~ → API route 가 INSERT/UPDATE 시 자동 호출

### 거의 안 해도 되는 일
5. 프로세스/코드 수정 — 제가 (AI 어시스턴트가) 해드림

## 📋 자주 묻는 질문 (FAQ)

### Q1. /register 이후 얼마나 기다려야 하나요?
**A**: 1~2분. AI 파싱 시간이 대부분. 사장님은 다른 일 하고 있으면 됨.

### Q2. 실패하면 재시도?
**A**: AI는 자동으로 2번 재시도 (llm-retry.ts). 그래도 실패 시 draft 저장 + 저한테 "다시 파싱해줘".

### Q3. 이미 등록된 상품 수정은?
**A**: `/admin/packages` 에서 해당 상품 클릭 → 필드 수정 → 저장. 자동으로 모바일 갱신됨.

### Q4. 사진이 안 뜨면?
**A**: attractions 테이블의 해당 관광지에 photos 필드 없음. `/admin/attractions` 에서 사진 수집 버튼 클릭.

### Q5. 같은 상품 2번 등록하면?
**A**: 자동 중복 감지 → 완전 동일이면 SKIP, 가격만 다르면 이전 버전 아카이브 + 신규 등록.

### Q6. 실수로 잘못 등록했어요
**A**: `/admin/packages` 에서 해당 상품 → status를 `archived`로 변경. 복구도 가능.

### Q7. 프로덕션에 바로 반영되나요?
**A**: 네. ISR revalidate가 자동 실행되어 즉시 고객에게 보임.

### Q8. 지역이 새로운 곳이면?
**A**: 첫 3번째 상품 등록 시 어셈블러(지역 전용 파서)가 자동 생성됨. 이후 더 정확한 파싱.
