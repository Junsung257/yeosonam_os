```bash

```

## ADR Lifecycle

- **Proposed** → Draft, under discussion
- **Accepted** → Approved and implemented
- **Rejected** → Considered but not adopted
- **Deprecated** → No longer relevant
- **Superseded** → Replaced by a newer decision (link to new ADR)
